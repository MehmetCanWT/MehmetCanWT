# MehmetCanWT Website 🚀

Bu proje `mehmetcanwt.xyz` için Next.js 15, PostgreSQL ve Docker kullanılarak geliştirilmiştir.

## 🛠 Teknolojiler
- **Framework:** Next.js 15 (App Router, TypeScript)
- **Database:** Prisma ORM & Native PostgreSQL
- **Deployment:** Docker & GitHub Actions
- **Port:** 3131

## 🐳 Docker Deployment
```bash
docker compose up -d --build
```

## 🔄 CI/CD
GitHub Actions ile `main` branchine atılan her commit otomatik olarak VDS sunucusuna deploy edilir.

## 🎮 Hakkımda
Anime izlemeyi ve oyun oynamayı seven bir geliştiricinin kişisel sayfası.
